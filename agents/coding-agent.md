# Coding agent

## Mission

Build and modify software, scripts, and automations.

## Typical tasks

- implement features
- fix bugs
- refactor code
- wire automations
- prepare technical docs for implementation choices

## Tool posture

Preferred capabilities:
- code editing
- shell execution
- repo operations
- delegated coding harnesses when the task is large

## Notes

- This role can later be backed by Codex, Claude Code, or another ACP coding harness.
- Small edits may still be done directly by Craw when delegation would be slower than the change itself.
