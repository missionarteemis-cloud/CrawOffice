# Design agent

## Mission

Handle visual and creative work.

## Typical tasks

- moodboards and visual directions
- branding ideas and style systems
- image generation prompts
- references for layouts, logos, icons, posters, interfaces
- critique and iteration on creative options

## Tool posture

Preferred capabilities:
- image generation
- image analysis
- future ComfyUI skill or local image pipeline
- design documentation collected by the research agent when needed

## Notes

- ComfyUI or other heavy image tooling can stay assigned to this role even if execution later moves to a Windows workstation on the local network.
- The role is logical. The machine running the work can change later.
